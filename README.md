# 🚀 AI Article Generator

Полностью автоматизированная система генерации и публикации статей с использованием LLM.

Проект позволяет создавать тематические ленты, автоматически генерировать уникальные статьи при помощи DeepSeek AI и публиковать их в Telegram и Яндекс Дзен по расписанию без участия человека.

# 📚 Управление тематиками

Для каждой тематики можно настроить:

- название;
- описание;
- количество публикаций в день;
- генерацию изображений;
- включение/отключение публикации.

---

# 📅 Планировщик

Проект автоматически распределяет публикации равномерно в течение дня.

Например:

```text
08:00
11:45
15:30
19:15
23:00
```

Scheduler периодически проверяет, пришло ли время очередной публикации.

---

# 🤖 Генерация статей

Во время генерации агент самостоятельно принимает решение, требуется ли обращение к поиску.

Используется механизм Function Calling.

Если статья требует актуальных данных:

- новости;
- статистику;
- цены;
- законодательство;
- рейтинги;

агент вызывает инструмент поиска.

В остальных случаях статья пишется исключительно на основании знаний модели.

---

# 🖼 Генерация изображений

Для тематик с включенной генерацией изображений агент дополнительно создает описание сцены.

После этого автоматически генерируется изображение через YandexART.

# 📤 Автопостинг

Поддерживаемые платформы:

- Telegram
- Яндекс Дзен

Telegram использует HTML-разметку.

Яндекс Дзен публикуется через Playwright с использованием сохраненной авторизованной сессии.
